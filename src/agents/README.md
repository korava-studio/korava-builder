# Agent Operating System (AgentOS)

AgentOS manages AI employees through lifecycle, communication, permissions, memory, and scheduling.

## Structure

- `core/` — base agent definitions and manager
- `communication/` — message bus for agent independence
- `permissions/` — permission guards
- `memory/` — agent memory stores
- `registry/` — registration and discovery
- `scheduler/` — task scheduling for agents
- `skills/` — skill assignments
- `lifecycle/` — lifecycle orchestration
- `employees/` — concrete agent roles
